# Introduction

Raw data collected from different sources is often **incomplete, inconsistent, or incorrect**. Before performing analysis, the data must be cleaned to improve its quality and reliability.

**Power Query** is Power BI's built-in **ETL (Extract, Transform, Load)** tool used for importing, cleaning, transforming, and preparing data.

### Definition

> **Data Cleaning** is the process of identifying and correcting errors, inconsistencies, duplicate records, and missing values in a dataset to improve its quality and accuracy.

---

# Why is Data Cleaning Important?

Poor-quality data can lead to:

* Incorrect reports
* Wrong business decisions
* Duplicate records
* Missing information
* Inaccurate calculations

### Example

Original Data

| Customer | City      | Sales |
| -------- | --------- | ----- |
| Ravi     | Bangalore | 25000 |
| Ravi     | Bangalore | 25000 |
| Anita    | *(Blank)* | 18000 |
| Rahul    | Mysore    | Error |

Problems:

* Duplicate record
* Missing city
* Error in Sales column

After cleaning

| Customer | City      | Sales |
| -------- | --------- | ----- |
| Ravi     | Bangalore | 25000 |
| Anita    | Bangalore | 18000 |
| Rahul    | Mysore    | 22000 |

The cleaned data is now suitable for analysis.

# 1. Handling Missing Values

## What are Missing Values?

Missing values are empty or null values where data is unavailable.

Example

| Name  | Department | Salary    |
| ----- | ---------- | --------- |
| Ravi  | Sales      | 45000     |
| Anita | HR         | *(Blank)* |
| Rahul | Finance    | 55000     |

The Salary for Anita is missing.

---

## Why Missing Values are Problematic

They can:

* Produce incorrect averages
* Affect charts and KPIs
* Cause errors in calculations
* Reduce data quality

---

## Methods for Handling Missing Values

### 1. Remove Missing Rows

Used when only a few records are incomplete.

**Power Query Steps**

* Select the column.
* Home → Remove Rows → Remove Blank Rows.

### Example

Before

| Name  | Salary    |
| ----- | --------- |
| Ravi  | 45000     |
| Anita | *(Blank)* |

After

| Name | Salary |
| ---- | ------ |
| Ravi | 45000  |

---

### 2. Replace Missing Values

Instead of deleting rows, replace blanks with meaningful values.

Example:

Blank Sales → 0

Blank City → "Unknown"

Power Query:

* Select column
* Transform → Replace Values

---

### 3. Fill Down

Copies the value from the previous row.

Example

Before

| Region    |
| --------- |
| South     |
| *(Blank)* |
| *(Blank)* |
| North     |

After Fill Down

| Region |
| ------ |
| South  |
| South  |
| South  |
| North  |

---

### 4. Fill Up

Copies values upward from the next available row.

Used when missing values appear above valid entries.

---

### Business Example

A retail store forgets to enter the city for some customers.

Instead of deleting customers, replace blanks with **"Unknown"** or use **Fill Down** if records belong to the same city.

# 2. Removing Duplicates

## What are Duplicate Records?

Duplicate records are identical rows appearing more than once.

Example

| Customer | Product | Amount |
| -------- | ------- | ------ |
| Ravi     | Laptop  | 60000  |
| Ravi     | Laptop  | 60000  |
| Anita    | Mobile  | 25000  |

The first two rows are duplicates.

---

## Problems Caused by Duplicates

* Double counting
* Incorrect sales totals
* Wrong customer counts
* Misleading reports

---

## Removing Duplicates in Power Query

Steps:

1. Select one or more columns.
2. Home → Remove Rows.
3. Click **Remove Duplicates**.

Power Query retains only the first occurrence.

---

### Before

| Customer | Sales |
| -------- | ----- |
| Ravi     | 10000 |
| Ravi     | 10000 |
| Anita    | 15000 |

### After

| Customer | Sales |
| -------- | ----- |
| Ravi     | 10000 |
| Anita    | 15000 |

---

### Business Example

An online shopping website accidentally stores duplicate orders.

Removing duplicates prevents customers from being counted twice.

# 3. Data Profiling Techniques

## What is Data Profiling?

Data profiling is the process of examining data to understand its quality, structure, completeness, and consistency.

Power Query provides built-in profiling tools.

---

# Objectives

* Detect missing values
* Detect errors
* Find duplicates
* Identify unusual values
* Verify data types

---

# Power Query Data Profiling Tools

## A. Column Quality

Shows:

* Valid values
* Errors
* Empty values

Example

| Column | Valid | Error | Empty |
| ------ | ----- | ----- | ----- |
| Sales  | 95%   | 2%    | 3%    |

Green indicates valid data, red indicates errors, and gray indicates empty values.

---

## B. Column Distribution

Displays:

* Distinct values
* Unique values
* Frequency distribution

Example

Department

Sales → 120

HR → 40

Finance → 25

This helps identify the distribution of categorical data.

---

## C. Column Profile

Displays detailed statistics such as:

* Minimum value
* Maximum value
* Average
* Median
* Standard deviation
* Number of distinct values

Example

Sales Column

Minimum = ₹5,000

Maximum = ₹2,50,000

Average = ₹48,000

---

### Benefits of Data Profiling

* Detects anomalies
* Improves data quality
* Identifies incorrect values
* Helps understand datasets before analysis

# 4. Error Detection and Correction

## What are Data Errors?

Errors occur when data contains invalid or inconsistent values.

Examples

* Text in numeric columns
* Incorrect dates
* Invalid data types
* Formula errors
* Missing records

---

## Types of Errors

### 1. Data Type Errors

Example

| Sales |
| ----- |
| 10000 |
| ABC   |

ABC is invalid because Sales should contain numbers.

---

### Correction

Change the column type to Number.

Replace invalid values or remove error rows.

---

### 2. Invalid Dates

Example

31/02/2026

February has only 28 or 29 days.

Correct the date or remove the record.

---

### 3. Formula Errors

Example

Division by zero

Replace the error with a valid value or null.

---

### 4. Outliers

Outliers are values that differ significantly from the rest of the data.

Example

Monthly Salary

45000

47000

49000

900000

The value ₹900000 may be an error and should be verified.

---

# Detecting Errors in Power Query

Power Query automatically highlights errors.

Steps:

1. Select the column.
2. Home → Keep Rows → Keep Errors.

This filters only rows containing errors.

---

# Correcting Errors

Methods include:

* Replace errors
* Remove error rows
* Change data types
* Edit incorrect values
* Re-import data if necessary

---

### Example

Before

| Sales |
| ----- |
| 12000 |
| Error |
| 15000 |

After Replace Errors

| Sales |
| ----- |
| 12000 |
| 0     |
| 15000 |
